Ibrowse User Guide

Ibrowse is the minimalistic browser built off of simplicity and ease-of-use.

This guide will help you understand Ibrowse and its advanced tools.


Section 1: Search Bar

The search bar can be used to search for anything on the internet, whether it's a url or a brownie recipe. Upon typing, you may notice a suggestions popup. This popup suggests bookmarks and commands when needed.

You may also notice the choice box with "Google" selected by default. This is the browser engine you can use for searching. You can choose Google, DuckDuckGo, Bing, and more beyond that.

In the corner with three dots (...) you can access the browser menu, to add things like passwords and bookmarks, create new tabs, and open new windows.

About Passwords & Bookmarks...

All passwords and bookmarks you create in Ibrowse are stored on your local machine. Ibrowse has no access to any cloud service or database, therefore, any user data (from Ibrowse) is stored in a config.json file.

Section 3: Commands

Previously mentioned in the search bar section, you can actually type commands into the search bar. You probably typed /help to access this page!

Listed below is all the commands that can be used in the search bar:

  • /help: access this page again
  • /welcome: open the welcome page shown on startup
  • /close: close the active tab
  • /exit: close Ibrowse entirely
