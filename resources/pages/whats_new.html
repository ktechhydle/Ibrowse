What's new in Ibrowse 2.0

Ibrowse 2.0 includes a complete ui overhaul, with a re-designed look to feel even more modern.

  • Search bar/Engine selector is unified into one widget
  • Search bar has a nice icon indicating the search box
  • Drag and drop urls into tab bar to open them
  • New "Lock Tab" feature to prevent access to certain tabs
  • Print pages (finally!)
  • Escape key animates the closing of quick search
  • Escape key to exit the search box
  • Ctrl+/ key to jump to the search box
  • Clear caches no longer requires a full restart

Ibrowse 2.0 has been heavily optimized, we actually saw loading times decrease by over 50%!

Tabs load and create faster, as we put some of the core logic into the main tab view. This means dialogs and actions are only created one time, compared to previous versions.

With the new lock tab feature, you can secure tabs if you need to step away from the desk. Co-workers can't sneak by and look at what you are browsing.

If a tab is locked, you can still stay inside of it and use it. It essentially means, as soon as a diffferent tab is opened, the locked one can't be accessed. It is manually unlocked by right clicking.
